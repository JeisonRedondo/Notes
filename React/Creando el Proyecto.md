## Bun 
 Bun es un entorno de ejecución para javascript y Typescript, gestor de paquetes , empaquetador y corredor de pruebas. Con este manejador crearemos la carpeta donde contendra nuestro proyecto.

## Creando el proyecto
```bash
bun create react-app gentleman-react
```
