El componente en pocas palabras es una función que se ejecuta y su resultado se incluye en el html que se va a renderizar. Por simple que parezca, es el proceso fundamental que se usa para creación y renderización de páginas en react.

Como tal el componente debe ser la mínima unidad logica posible

## Archivos (jsx, tsx)
Los archivos terminados en "x" son interpretados en react como archivos con html y logica de javascript, esto permite una interacción entre las dos partes que facilita la creación de las reacciones de la aplicación a varias acciones del usuario.\

### Componente Tonto
Es aquel que se usa mayormente para definir le estilo o un patrón simple, se usa principalmente para presentar una ayuda visual al usuario
### Componente inteligente
Es aquel que carga con estados y lógica de la aplicación
