# UseCallback
Se utiliza para guardar una instancia de una función, hace que un hijo no renderice.

## Ejemplo
    Supongamos que tienes un número de telefono al que llamas con frecuencia, en vez de marcarlo continuamente lo vamos a almacenar en los contactos del teléfono. A menos que el número cambie siempre se utiliza el mismo contacto.
