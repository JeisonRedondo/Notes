## Composición de Componentes
- Crear componentes que son eficientes y adaptables
- Composición que permita entender facilmente su funcion y su estructura
## Colocación de Estado
- Prioridad al acceso eficiente de los estados
- La cercania o la disposición de los componentes objetivo debe ser clara
- Componentes con funcionalidad y ui deben tener claras divisiones

## ??
- Cuantos manejadores de estados puedes usar en react de manera nativa?
React maneja el estado de forma nativa principalmente con useState para estado local simple y useReducer para estado local más complejo, además de la Context API, que no gestiona estado por sí sola pero permite compartirlo globalmente al combinarse con useState o useReducer; en resumen, React tiene dos manejadores de estado reales y un mecanismo nativo para distribuir estado entre componentes.
