Usar prompts largos muchas veces puede llevarnos a perdernos si no estan bien estructurados, pero la estructura que nos permite visualizar a nosotros de manera apropiada no servira para un llm.Usar XML nos permite transmitir uan esructura al llm y darle claridad sobre la información tal como uan buena estructura servira para nosotros

## XML?
Es un lenguaje basado en etiquetas que nos permite separar el contenido dentro de diferentes etuiqeutas, esto da especificidad al contenido del prompt y ayuda al llm al ingresar la información.

## ¿Como es una etiqueta XML?
Estas solo tienen una regla:
- Cada etiqueta que se abre, se cierra

Al cerrarla nos permite definir esa información, cada etiqueta puede ser nombrada como se desee pero lo importante es darle contexto, pues esta tambien hace parte del prompt.
