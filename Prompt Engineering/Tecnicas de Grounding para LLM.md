# Validar la información

## Como?
- Usamos palabras que definan el alcance de busquda y las referencias
- Le pedimos que nos muestre de manera explicita la parte de donde se referencia estas respuestas
- Buscamos estas referencias con el fin de validar la información

## Por que?
- Necesitamos mantener un control de la informacióny su veracidad
- Al controlar las referencias podremos obtener respuestas acorde a la tematica que deseamos resolver
