# Definir mis objetivos (Prompt Shadow)
- Permite saber la respuesta a la que quieres que el llm llegue
- Ayuda a calibrar los limites
- Nos provee de una salida esperada

## Como?
- Podemos empezar con un enunciado general
- Vamos dando un contexto cada vez mas cercano
- Le damos informacion que permita deducir la respuesta mas acorde
- Una vez que obtenemos el resultado esperado, le preguntamos al llm que prompt hubiera dado este resultado desde el principio
