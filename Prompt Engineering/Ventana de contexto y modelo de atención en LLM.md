# Ventana de contexto y Modelo de atención

## Ventana de contexto?
- Es la información  que se ah dispuesto en el chat, con el fin de generar un punto de partida y controlar las respuesta del LLM
- Incluye el prommpt y tambien las subsecuentes respuestas

## Modelo de atención
- Refiere a la atención que mantiene el modelo sobre la información que se ha dispuesto para que se genere la respuesta
- Tiene un limite para cada llm
- Es una caracteristica a la cual se le debe poner cuidado y optimizar


## Despues del limite, que pasa?

- Suele ocurrir que la información y las respuesta que se reciben no sean correctas
- Al pasar de este limite, el LLM suele "olvidar" cierta información y no tenerla en cuenta en las respuestas
