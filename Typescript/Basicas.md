# Las basicas

## Errores mas comunes en JS
- Entre los mas comunes estan los tipos de las diferentes entidades.
- Al no se visualizables podemos desconocer sus caracteristicas.
- Muchas veces sino siempre requiere de la ejecución del codigo para saber
las propiedades del mismo.

## Tipado Dinamico
- Al usar tipado estatico, podemos evitar errores de tipado muy facilmente.
- Ayudara a evitar la perdida de tiempo en la resolución de errores de código.
- Ayuda a evidenciar diferentes errores que pudimos pasar por alto sin tsc

## Ejecución de Tsc
- Primero instalamos el compilador de tsc:
```bash
npm install -g typescript

```
--> Esto nos permitira la ejecución de ts

- Luego al tener el archivo editado y guardado, podemos ejecutarlo asi: 
```bash
tsc "nombre del archivo"
```
--> Así podras crear un archivo de js con el contenido escrito en ts traducido a js, pero filtrado antes de para evitar cierto errores
