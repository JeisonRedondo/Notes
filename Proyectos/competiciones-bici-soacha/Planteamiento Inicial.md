# Creación de planteamiento inicial

## Idea General

Es una pagína que tendra como punto principal mostrar los proximos eventos de carreras en bicicleta en soacha,
como se torna el ambiente, noticias de bicicletas y mas.

## Caracteristicas Basícas

- Un carrusel que contenga, las noticias mas importantes del momentoy la publicidad de los eventos mas relevantes.
- Un main que contenga blogs de las carreras pasadas.
- Redes y pagínas más relevantes de las carreras en soacha.
- Poder ver las posiciones de los corredores en soacha dado sea el caso.

## Caracteristicas fuera del objetivo

- Visualización del recorrido de las carreras
- Referencias de los corredores, redes y ultimos twits
- Lista de puntos en base de las ultimas carerras

## Tecnologias a usar
    - react
    - css
