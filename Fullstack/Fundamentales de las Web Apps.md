# Fundamentos de las Web Apps
## Reglas del Desarrollador
1. Siempre manten abierta la consola de desarrollador en el navegador.
(Mantendremos en uso la parte de network, desactivada la de preserve log y activada la de disable cache)


## Headers -> General
Nos muestra la información referente a la request que se le hizo a la página en la UrlRequest, la cuál salio correcta,
esto lo podemos diferenciar gracias al StatusCode el cuál es 200.

## Headers -> Response Headers
- Content-Lenght: Nos muestra el tamaño de la respuesta en bytes.
- Content-Type: Nos muestra el tipo de formato y/o archivo de la respuesta.

## Headers -> Response Tab
- Nos muestra la página en HTML plano.
